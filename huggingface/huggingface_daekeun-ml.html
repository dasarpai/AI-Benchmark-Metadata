### 1. 전송선의 특성 임피던스 (Characteristic Impedance, \( Z_0 \)) 계산 특성 임피던스 \( Z_0 \)는 다음 공식으로 계산됩니다: \[ Z_0 = \sqrt{\frac{R + j\omega L}{G + j\omega C}} \] 주파수 \( f \)가 주어지지 않았으므로, 일반적인 경우를 고려하여 \( G = 0 \)인 경우로 단순화할 수 있습니다: \[ Z_0 = \sqrt{\frac{R + j\omega L}{j\omega C}} \] 여기서 \( \omega = 2\pi f \)입니다. 주파수를 고려하지 않고, 저항과 인덕턴스, 캐패시턴스만으로 특성 임피던스를 계산하면: \[ Z_0 = \sqrt{\frac{R}{j\omega C}} \] 그러나, 주파수 \( f \)가 주어지지 않았으므로, 주파수에 무관한 특성 임피던스를 계산할 수 없습니다. 주파수를 주어야 정확한 계산이 가능합니다. ### 2. 전송선의 전파 속도 (Propagation Velocity, \( v_p \)) 계산 전파 속도 \( v_p \)는 다음 공식으로 계산됩니다: \[ v_p = \frac{1}{\sqrt{LC}} \] 주어진 값들을 대입하면: \[ L = 2 \mu H/m = 2 \times 10^{-6} H/m \] \[ C = 50 pF/m = 50 \times 10^{-12} F/m \] 따라서: \[ v_p = \frac{1}{\sqrt{(2 \times 10^{-6})(50 \times 10^{-12})}} \] \[ v_p = \frac{1}{\sqrt{1 \times 10^{-16}}} \] \[ v_p = \frac{1}{10^{-8}} \] \[ v_p = 10^8 \, m/s \] ### 3. 주파수 \( f \)가 1 MHz일 때, 전송선의 파장 (Wavelength, \( λ \)) 계산 파장은 다음 공식으로 계산됩니다: \[ \lambda = \frac{v_p}{f} \] 주파수 \( f = 1 \, MHz = 1 \times 10^6 \, Hz \)이고, 전파 속도 \( v_p = 10^8 \, m/s \)이므로: \[ \lambda = \frac{10^8}{1 \times 10^6} \] \[ \lambda = 100 \, m \] ### 4. 전송선의 반사 계수 (Reflection Coefficient, \( Γ \)) 계산 반사 계수 \( Γ \)는 다음 공식으로 계산됩니다: \[ \Gamma = \frac{Z_L - Z_0}{Z_L + Z_0} \] 여기서 \( Z_L = R_L = Z_0 \)이므로: \[ \Gamma = \frac{Z_0 - Z_0}{Z_0 + Z_0} \] \[ \Gamma = \frac{0}{2Z_0} \] \[ \Gamma = 0 \] 따라서, 반사 계수 \( Γ \)는 0입니다.

전송선로의 4가지 주요 파라미터는 다음과 같습니다: 1. **저항 \(R\)**: 전송선로의 도체가 전류를 흐를 때 발생하는 저항입니다. 저항은 전력 손실을 일으키며, 전송선로의 효율을 감소시킵니다. 2. **인덕턴스 \(L\)**: 전송선로의 도체 주위에 형성되는 자기장에 의해 발생하는 인덕턴스입니다. 인덕턴스는 전송선로의 전압과 전류 간의 위상차를 유발하며, 고주파 신호의 전송에 영향을 미칩니다. 3. **정전 용량 \(C\)**: 전송선로의 도체 간에 형성되는 전기장에 의해 발생하는 정전 용량입니다. 정전 용량은 전송선로의 전압과 전류 간의 위상차를 유발하며, 신호의 전파 속도에 영향을 미칩니다. 4. **컨덕턴스 \(G\)**: 전송선로의 절연체가 완벽하지 않아 발생하는 누설 전류에 의한 컨덕턴스입니다. 컨덕턴스는 전력 손실을 일으키며, 전송선로의 효율을 감소시킵니다. 이 파라미터들은 전송선로의 특성 임피던스 \(Z_0\)와 전파 상수 \(\gamma\)를 계산하는 데 사용됩니다. **특성 임피던스 \(Z_0\)**: \[ Z_0 = \sqrt{\frac{R + j\omega L}{G + j\omega C}} \] - \(R\)과 \(L\)이 증가하면 \(Z_0\)가 증가합니다. - \(G\)와 \(C\)가 증가하면 \(Z_0\)가 감소합니다. **전파 상수 \(\gamma\)**: \[ \gamma = \alpha + j\beta = \sqrt{(R + j\omega L)(G + j\omega C)} \] - \(R\)과 \(G\)가 증가하면 감쇠 상수 \(\alpha\)가 증가하여 신호 감쇠가 커집니다. - \(L\)과 \(C\)가 증가하면 위상 상수 \(\beta\)가 증가하여 신호의 위상 변화가 커집니다. 이와 같이, 전송선로의 저항, 인덕턴스, 정전 용량, 컨덕턴스는 전송선로의 성능에 중요한 영향을 미치며, 특성 임피던스와 전파 상수를 통해 전송선로의 전기적 특성을 결정합니다.

                  Korean GLAN (Generalized Instruction Tuning) Instructions Dataset

                  What is GLAN?

                  Catastrophic forgetting, also known as catastrophic interference, occurs during SLM/LLM fine-tuning when a model trained on new data overwrites the knowledge it previously acquired, leading to a significant drop in performance on earlier tasks. This issue is particularly prominent in scenarios where the model needs to adapt to diverse and potentially conflicting data distributions without losing its ability to perform well on initial tasks.

                  GLAN (Generalized Instruction Tuning) addresses catastrophic forgetting by leveraging a systematically curated taxonomy of human knowledge to generate synthetic, diverse, and comprehensive instruction datasets. Specifically, GLAN uses a pre-curated taxonomy of human knowledge and capabilities to generate large-scale, diverse instruction data across various disciplines. This taxonomy mirrors the systematic structure of human education, breaking down knowledge into fields, sub-fields, and distinct disciplines, which are then used to design syllabi for different subjects. These syllabi detail key concepts for each class session, enabling the generation of diverse and comprehensive instructions. Here is GLAN's key features:

                  • Taxonomy Creation: A detailed hierarchy of human knowledge is built using LLMs and human verification.
                  • Subject and Syllabus Generation: LLMs generate a list of subjects and detailed syllabi, breaking down subjects into class sessions with key concepts.
                  • Instruction Generation: Homework questions and their answers are generated based on the syllabi, ensuring broad coverage and diversity.

                  Overview

                  This dataset is generated using the GLAN approach, inspired by the paper described in the paper. The dataset was created by running the hands-on reference code provided in the implementation by the author.

                  Dataset Details

                  • Number of records: 300K (303,581)
                  • Language: Korean
                  • Purpose: This dataset is designed for fine-tuning task and for question answering, but not limited to question answering.

                  Dataset Creation

                  The dataset was created using a synthetic generation approach based on instructions and questions. The generation process leverages Microsoft Phi-3.5-MoE to create diverse and high-quality examples that are valuable for training and evaluating NLU models.

                  Features

                  The dataset contains the following features:

                  • question: The instruction or question prompt given to the model.
                  • answer: The generated response based on the instruction.
                  • subject: A subject derived from a taxonomy. A syllabus is generated from this subject.
                  • level: The difficulty level of the course. For a first-year undergraduate course, think of it as level 100. (e.g., 100, 200, 300, ...)
                  • subtopics: These are subtopics related to the subject.
