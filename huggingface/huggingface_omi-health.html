              Dataset Card for Synthetic Medical Dialogues and SOAP Summaries

              Dataset Description

              Abstract

              This dataset consists of 10,000 synthetic dialogues between a patient and clinician, created using the GPT-4 dataset from NoteChat, based on PubMed Central (PMC) case-reports. Accompanying these dialogues are SOAP summaries generated through GPT-4. The dataset is split into 9250 training, 500 validation, and 250 test entries, each containing a dialogue column, a SOAP column, a prompt column, and a ChatML-style conversation format column.

              Data Splits

              • Training: 9250
              • Validation: 500
              • Test: 250

              Access

              The dataset is available under specific conditions:

              • Usage is permitted for any purpose, commercial or otherwise, by Omi, provided that permission is obtained from the original authors of the NoteChat dataset for any further use.

              Using the Dataset

              This dataset is ready for use in tasks related to natural language understanding, dialogue generation, and medical documentation automation. It's specifically structured to aid in developing AI models that can generate and understand clinical conversations and their summarized forms. However, please note that the dialogues and summaries are synthetically generated and not designed for direct clinical application without appropriate guardrails and validation.

              Acknowledgements

              This dataset was created using the GPT-4 model applied to the NoteChat dataset. We acknowledge the creators of the NoteChat dataset for their original work and thank them for their permission to use it in creating this dataset. For more information, please refer to their paper: NoteChat Paper.
