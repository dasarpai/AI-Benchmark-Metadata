                              Dataset Card for CIFAR-100

                              Dataset Summary

                              The CIFAR-100 dataset consists of 60000 32x32 colour images in 100 classes, with 600 images per class. There are 500 training images and 100 testing images per class. There are 50000 training images and 10000 test images. The 100 classes are grouped into 20 superclasses. There are two labels per image - fine label (actual class) and coarse label (superclass).

                              Supported Tasks and Leaderboards

                              • image-classification: The goal of this task is to classify a given image into one of 100 classes. The leaderboard is available here.

                              Languages

                              English

                              Data Fields

                              • img: A PIL.Image.Image object containing the 32x32 image. Note that when accessing the image column: dataset[0]["image"] the image file is automatically decoded. Decoding of a large number of image files might take a significant amount of time. Thus it is important to first query the sample index before the "image" column, i.e. dataset[0]["image"] should always be preferred over dataset["image"][0]
