To change the color of the thumb on a seek bar when it is held, you can create a custom drawable for the thumb and set it as the thumb of the seek bar. 1. Create a new drawable xml file in your project's res/drawable folder. For example, you can name it seekbar_thumb.xml. 2. In the seekbar_thumb.xml file, add the following code to create a custom shape and set its color: ``` <shape xmlns:android="http://schemas.android.com/apk/res/android"> <solid android:color="@color/blue" /> </shape> ``` Replace "@color/blue" with the blue color that you want to use. 3. Add the following code to the same seekbar_thumb.xml file to set the size of the thumb: ``` <selector xmlns:android="http://schemas.android.com/apk/res/android"> <item android:state_pressed="true"> <shape android:shape="oval"> <size android:width="40dp" android:height="40dp" /> </shape> </item> <item> <shape android:shape="oval"> <size android:width="25dp" android:height="25dp" /> </shape> </item> </selector> ``` Adjust the values of the width and height attributes to set the size of the thumb as per your requirement. 4. In your layout xml file where you have the seekbar, set the thumb of the seekbar to the custom drawable using the android:thumb attribute: ``` <SeekBar android:id="@+id/mySeekBar" android:layout_width="match_parent" android:layout_height="wrap_content" android:thumb="@drawable/seekbar_thumb" /> ``` That's it! Now when the thumb of the seekbar is held, it should turn blue.
